# Задачи: Optional binding

## Задача 1 — второе слово
Напиши функцию `printSecondWord(_:)`, которая принимает `String?` и печатает **второе слово** (индекс 1).  
Если строки нет или слов меньше двух — напечатай `"no second word"`.

## Задача 2 — среднее арифметическое
Напиши функцию `calculateAverage(scores:) -> Double?`, которая:
- отбрасывает `nil`
- вычисляет среднее
- возвращает `nil`, если после отбрасывания массив пуст

## Задача 3 — глубокий опционал
Напиши функцию `sumEven(in:) -> Int`, которая:
- принимает `[Int?]?`
- убирает `nil` на обоих уровнях
- суммирует чётные числа
- если массива нет или чётных нет — возвращает `0`
