
# 📝 Задачи на цепочки методов (filter, map, sorted, compactMap, reduce)

## Задача 1
Дан массив строк `["swift", "xcode", "playground", "debug", "compiler"]`.
Оставь слова длиннее 5 символов, приведи к верхнему регистру, отсортируй по алфавиту. Используй `filter`, `map`, `sorted`.

## Задача 2
Дан массив чисел `[3, 7, 0, -5, 8, -2, 10, -1, 4]`.
Оставь чётные числа и возведи их в квадрат. Используй `filter`, `map`.

## Задача 3
Дан массив опциональных целых чисел `[5, nil, 12, 3, nil, 8, 10, nil, 1]`.
Отбрось `nil`, оставь только положительные числа. Верни строку в формате `"Positives: N, summa: S"`.  
Если после отбрасывания `nil` массив пуст → `"No data"`.  
Если нет положительных чисел → `"No positive numbers"`.  
Используй `compactMap`, `filter`, `count`, `reduce`.
