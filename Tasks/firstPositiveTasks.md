# Задачи: поиск первого элемента

## Задача 1
Напиши функцию `firstPositive(in:) -> String`, которая:
- принимает `[Int]?`
- если массив `nil` или пуст → `"No data"`
- если нет положительных чисел → `"No positive numbers"`
- иначе → `"First positive: <значение>"`
